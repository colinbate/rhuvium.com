Group Hug Beta

Group Hug is an easy to use Python-based RSS/Atom feed aggregator. If you have multiple weblogs and you would like to combine them all into one location — either for public display or simply for personal convenience, than this is the right tool for you. Perhaps you would like to aggregate all of the weblogs of your friends or colleagues into one location for a centralized information source. Whatever your reasons, combining multiple feeds into one site has never been this easy before.

Built on top of Mark Pilgrim's feedparser.py it handles all the various versions of RSS and Atom that are in use at the moment. It also uses urlgrabber to make sure that we don't download the whole feed if it hasn't been modified.

Unfortunately, the software isn't available for download at the moment. It is still undergoing proper testing before being released for public consumption. If you found this page from a User Agent string, then likely someone is aggregating your feed using Group Hug.

Features

  • Parses RSS and Atom feeds (all versions).
  • Combines multiple feeds into one list.
  • Standards compliant output.
  • Creates output in two separate files that can be included into site templates.
  • Respects 304 Response codes.
  • Caches most recent feed locally.
  • Can be set to run periodically or as a 'ping' from weblog submissions.
  • Very easy to set up.
