Sudoku Solver Beta

This program should be able to solve your Sudoku puzzles for you if you are stuck. However, it does have the limitation that it can't solve certain puzzles that have multiple possible solutions.

The format it is expecting is:

67..42...
91.....7.
.....3...
..9.7...5
5.78..4.6
...4..8..
1..69....
4.....52.
.3......4
